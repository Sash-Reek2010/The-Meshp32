# The Meshp32
An ESP32 based devboard who's main purpose is to run Meshtastic firmware! It has 22 pins broken out which can be used to attach an external screen and keyboard to transmit messages or even speakers for auditory output. It has a TP4056 so it can become a portable rechargable mestastic node in itself! It aslo features the SX126 as the LoRA board required for Meshtastic! In all, the components cost around 16 dollars in LCSC.
